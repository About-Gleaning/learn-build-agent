读取本地文件系统中的文件。你可以使用此工具直接访问任意文件。

请假设该工具能够读取机器上的所有文件。如果用户提供了某个文件路径，请假设该路径是有效的。读取一个不存在的文件也是可以的；工具会返回错误信息。

使用规则：
- 必须传入绝对路径参数 `file_path`，不支持相对路径
- 你也可以选择性地指定行偏移量（offset）和读取数量（limit）（对于长文件尤其方便），但通常建议不传这些参数，以便读取整个文件
- 不传 `limit` 时，默认返回从 `offset` 开始的全部文本内容
- 如果文件存在但内容为空，会返回成功结果，并明确提示“文件存在，但内容为空。”
- 你可以在单次响应中调用多个工具。对于可能有帮助的多个文件，最好主动成批读取
- 文件不存在、越界访问或参数非法时，会返回结构化错误信息
