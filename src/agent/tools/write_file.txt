把给定完整文本整文件写入目标文件；如果文件已存在，会被新内容完全覆盖；如果不存在，会直接新建。

使用要求：
- `filePath` 正式建议传绝对路径；若传相对路径，会按当前工作区根目录解析。
- 本工具是整文件覆盖写入，不适合只修改几行的小变更；局部改动优先使用 `edit_file`。
- 若目标文件已存在，写入前必须先用 `read_file` 读取同一文件；否则会失败。
- 当前工具会校验“最近一次读取”之后文件是否又被修改过；若文件已变化，必须重新读取后再写入。
- 写入前会校验工作区边界与路径合法性，禁止越权写到工作区外。

返回结果：
- `title`：相对于工作区的文件路径；若不是工作区内文件，则回退为绝对路径
- `output`：执行结果说明；当前未启用 diagnostics 时会显式提示
- `metadata.filepath`：最终写入文件的绝对路径
- `metadata.exists`：写入前文件是否已存在
- `metadata.diagnostics`：当前固定返回空数组，预留后续接入各语言 LSP
- `metadata.diagnostics_status`：当前固定为 `not_enabled`

注意：
- 当前仅支持文本写入，不负责局部 diff 或二进制写入语义。
- “写入成功”只代表文件落盘成功，不代表代码一定正确；未来接入 language service 后会在 diagnostics 中补充语法/类型诊断。
