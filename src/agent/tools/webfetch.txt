- 从指定 URL 获取内容
- 输入参数为一个 URL，以及可选的格式参数
- 获取该 URL 的内容，并转换为所请求的格式（默认是 markdown）
- 以指定格式返回内容
- 当你需要检索并分析网页内容时，使用这个工具

使用说明：
    - **重要：** 如果存在另一个在网页抓取方面能力更强、更贴合任务、或限制更少的工具，应优先使用那个工具，而不是这个工具。
    - URL 必须是完整且有效的 URL
    - `http` URL 会自动升级为 `https`
    - 格式选项包括：`"markdown"`（默认）、`"text"`、`"html"`
    - 这个工具是只读的，不会修改任何文件
    - 如果内容非常大，返回结果可能会被摘要化
